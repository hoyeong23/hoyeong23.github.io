# Personal Homepage

GitHub Pages로 배포할 수 있는 개인 홈페이지 시작 템플릿입니다.

## 파일 구성

- `index.html`: 메인 페이지
- `styles.css`: 스타일
- `script.js`: 간단한 스크롤 애니메이션

## GitHub Pages로 배포하는 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더의 파일을 저장소에 업로드합니다.
3. 저장소 `Settings` -> `Pages`로 이동합니다.
4. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
5. 브랜치는 `main`, 폴더는 `/ (root)`를 선택하고 저장합니다.
6. 잠시 후 `https://사용자이름.github.io/저장소이름/` 주소로 접속할 수 있습니다.

사용자 이름 저장소를 `사용자이름.github.io`로 만들면 `https://사용자이름.github.io/` 주소로 바로 연결됩니다.
