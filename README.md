# Hoyeong Yeo Homepage

Google Sites에 있던 핵심 텍스트 정보를 바탕으로 다시 구성한 GitHub Pages용 개인 홈페이지입니다.

## 포함한 내용

- 연구자 소개와 소속
- 학력 정보
- 연구 주제
- 국제/국내 학회 발표 목록
- 수상 및 연구 과제
- 연락처와 GitHub 링크

## 파일 구성

- `index.html`: 메인 페이지
- `styles.css`: 스타일
- `script.js`: 스크롤 등장 애니메이션

## GitHub Pages 배포 방법

1. GitHub에서 새 저장소를 만듭니다.
2. 저장소 이름은 `hoyeong23.github.io`를 추천합니다.
3. 이 폴더 파일들을 저장소에 업로드합니다.
4. 저장소 `Settings` -> `Pages`로 이동합니다.
5. `Deploy from a branch`를 선택합니다.
6. 브랜치는 `main`, 폴더는 `/ (root)`를 선택합니다.
7. 잠시 후 `https://hoyeong23.github.io/` 주소로 접속할 수 있습니다.
